**Использованные команды для первичной настройки Ubuntu 22.04**

```python
cat /etc/*release
apt-get upgrade
apt-get update
apt-get install -y vim htop git curl wget
```

Настройка входа по ssh

```python
vim /etc/ssh/sshd_config
service ssh restart
vim ~/.ssh/authorized_keys
chmod 700 ~/.ssh
chmod 600 ~/.ssh/authorized_keys
```
